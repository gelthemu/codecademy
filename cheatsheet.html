HTTP Status Codes

This table has some common HTTP status codes, each with its respective meaning.


HTTP status codes are crucial for understanding the outcome of a web request.

Status Code Name Meaning
200 OK The request was successful.
301 Moved Permanently The resource has been permanently moved to a new URL.
302 Found The resource is temporarily located at a different URL.
400 Bad Request The server couldn't understand the request.
401 Unauthorized Authentication is required to access the resource.
403 Forbidden Access to the resource is forbidden.
404 Not Found The requested resource was not found.
500 Internal Server Error A generic server error occurred.
502 Bad Gateway The server, while acting as a gateway, received an invalid response.
503 Service Unavailable The server is currently unavailable.
504 Gateway Timeout The server, while acting as a gateway, didn't receive a timely response.

HTTP status codes provide essential information about the success or failure of web requests, helping both clients and servers communicate effectively during web interactions.
